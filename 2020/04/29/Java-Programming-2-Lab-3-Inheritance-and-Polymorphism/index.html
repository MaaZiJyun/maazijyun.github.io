Java Programming 2 Lab 3 Inheritance and Polymorphism
2020-04-29

Question:

Create a hierarchy of classes for a set of vehicles using the following class names: Vehicle, Bicycle, Car, Motorbike, airplane.

The common methods for these vehicles are move(), printName(), printType() and printSeat(). The move() method prints out the distinct way on how a specific vehicle moves. For instance, the bicycle will be ‘paddling…’, the airplane will be ‘flying…’. The printName() method prints out registration number of vehicle (e.g. MH371). The printType() method prints the class name. The printSeat() method prints out how many seats available for passengers and drivers for a specific vehicle. The seatSize is a common attribute for all vehicles.

Create a test class to apply the Polymorphism concept using the following scenario.

Create an array of vehicles to store different types of vehicle instances. This array must be accessible by 2 methods: test1() and test2().

The test1() method shall create different vehicle instances including the generic vehicle and store them in the array. Then, use a For loop to loop through the array to call all the callable instance methods. Take the screenshot of your execution result (whole ide screen) for submission later.

Modify your airplane class by adding a method call autopilot() to toggle autopilot mode. Calling the autopilot() method will print out “airplane autopilot [ON/OFF].

In the test2 method, use a For loop to loop through the array to toggle the autopilot mode for airplanes only.

Take the screenshot of your execution result (whole ide screen) for submission later.

For submission, you need to submit the .java files together with 2 screenshots. DO NOT compress/zip the files.
