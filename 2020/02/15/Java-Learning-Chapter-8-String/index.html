8. Finding a Character or a Substring in a String

8.1. Simple searching

"Welcome to Java".indexOf('W'); returns 0.

Return the index of the first character in the string that matches the specified character.

"Welcome to Java".indexOf("come"); returns 3.

Returns the index of the first character in the string starting from the specified index that matches the specified character.

if there’s no searching letter in String

"Welcome to Java".indexOf('x'); returns -1.

8.2. Accurate searching

"Welcome to Java".indexOf('o', 5) returns 9.

"Welcome to Java".indexOf("Java", 5) returns 11.

9. Convert

9.1. Character and Numbers to Strings

The String class provides several static valueOf(); methods for converting a character, an array of characters, and numeric values to strings.

For example, to convert a double value to a string, use String.valueOf (5.44);. The return value is a string that consists of characters ‘5’, ‘.’, ‘4’, and ‘4’.
