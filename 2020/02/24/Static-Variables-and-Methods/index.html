Static Variables and Methods

2020-02-24

1. Static Variables

Static variables are shared by all the instances of the class.

But, you cannot use method like getPI() and setPI(double PI) because these are not static, so you need create object.

2. Static Methods

Static methods are not tied to a specific object.

  1. static
  1. not static

If the method is static, it refers to method that belongs to the class. So people can use the method without creating object.

If the variable is not static:

You cannot use the variable directly. So you need create object. Constants c1 = new Constants();

3. Static Constants

Static constants are final variables shared by all the instances of the class.

  1. non-static
  1. static

If the constants is static, it refers to constants that belongs to the class. So people can use the variable without creating object.

If the constants is not static:

You cannot use the constants directly. So you need create object. Constants c1 = new Constants();
