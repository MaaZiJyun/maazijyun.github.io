How to draw a Box Plot
2020-02-28

1. Read the data and sort numbers

So from the information that we can get the maximum and minimum of this data(question).

Max = 25 and Min = 4

2. Compute the Q1, Medium, Q3, IQR, LF and UF

1. Q1

Q1: 0.25(n + 1) n means the number of data

Imagine set of information is an array named “data” (this array’s index begin from 1 not 0)

From the information we know n = data.length = 10

Location of Q1 = 0.25 * (10 + 1) = 2.75

Last-Location < 2.75 < Next-Location

Thus, Last-Location = 2, Next-Location = 3;

Q1 = data[Last-Location] + 0.25 * (data[Next-Location] - data[Last-Location])

so, Q1 = 8 + 0.25(9 - 8) = 8.25

2. Medium

Medium: 0.5(n + 1) n means the number of data

Location of Medium = 0.5 * (10 + 1) = 5.5

Medium = 11 + 0.5(13 - 11) = 12

3. Q3

Same way to Q3:

Q3: 0.75(n + 1) n means the number of data

Location of Q3 = 0.75 * (10 + 1) = 8.25

so, Q1 = 8 + 0.25(9 - 8) = 19.5

4. IQR

IQR = Q3 - Q1

= 19.5 - 8.25 = 11.25

5. LF and UF

LF = Q1 - 1.5(IQR)

UF = Q3 + 1.5(IQR)

So, LF = 8.25 - 1.5(11.25) = -5.875

UF = 19.5 + 1.5(11.25) = 33.125

6. Draw Box Plot

THIS IS THE EXAMPLE OF HOW TO DRAW A BOX PLOT

THE DATA WE MIGHT USE:

MAX = 25

MIN = 4

Q1 = 8.25

Q3 = 19.5

MEDIUM = 12

LF = -5.875

UF = 33.125
