How to Make a Simple Online Calculator

2020-10-29

Preparation

Requested Application

  1. JDK 1.8
  2. Oracle Xe Version 18c
  3. Tomcat Apache 9
  4. Oracle SQL Developer 19.2
  5. Eclipse EE for Java Developers

Create a Maven Project

Step 1: Open Eclipse

Step 2: Launch it and wait for a while

Step 3: Go to FILE, then new MAVEN PROJECT. or ALT+CTRL+N

Step 4: Don’t need to change first page, next.

Step 5: Find a filter with group id org.apache.maven.archetypes and artifact id maven-archetype-webapp, select it and next.

Step 6: Type in the Group Id (mine is: com.ssk3408) and Artifact Id (mine is: MVC-Calculator). Then Finish.

Step 7: You will get 1 error and 2 exception in Markers

Step 8: Right click on the MVC-Calculator in Project Explorer, and go to BUILD PATH and go to configure build path

Step 9: Change JDK to 1.8 in Java Compiler, Apply

Step 10: Change JDK to 1.8 in Project Facets, Apply

Step 11: add your Tomcat in Targeted Runtimes

Step 12: Select items named Maven Dependencies, Apache Tomcat v9.0 and JRE System Library in Java Build Path. apply and close it

Step 13: The errors disappeared

(Before you create the project, shut down the tomcat)

Problem and solution

‘Publishing to Tomcat v9.0 Server at localhost…’ has encountered a problem.

If you get this error when you try to run the program, it might because you didn’t shut down the tomcat before you run it.

If you already shut down the Tomcat but still doesn’t work, go to your Tomcat’s file named Server where you installed your Tomcat.

The path in my computer for reference: D:\APACHE\Server\conf

Then find the file named server.xml and open it with Notepad

Ctrl + F to find the keyword: Shutdown and change port from -1 to 1
