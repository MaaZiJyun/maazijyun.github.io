2020-10-24

Add JavaFX to Eclipse

  1. Open Eclipse.
  2. Click “Help“ on the top left selection bar.
  3. Click “Eclipse Marketplace“
  4. Waiting for loading and input “fx“ in searching bar.
  5. Then install it.
  6. And the Eclipse would be restarted.
  7. Then you may see the “JavaFX” in project creating category.
  8. However, you will see there’re a lot of errors in the sample javaFX program. Now it’s the time to download the JavaFX files.
  9. Go to website: https://gluonhq.com/products/javafx/, and download one suitable to yours.
  10. Unzip the file you download, and remember the PATH where you unzipped it.
  11. Then we back to Eclipse. Click “Windows” on the top left selection bar.
  12. Click “Preferences“
  13. Go to User Libraries
  14. New a Library and name it whatever you want. (like JavaFX)
  15. Then select the new library, and click Add External JARs…
  16. Fine and select the files you unzipped. Apply and close it.
  17. Right click the project in Project Explorer
  18. Build Path
  19. Go to Libraries, and select Classpath. Add Libraries.
  20. User Library -> JavaFX
  21. Apply and close
  22. The errors and red lines are gone. But there’s still a problem when you run the file.
  23. Then go to the Arguments in Run Configurations.
  24. Edit VM arguments:

D:\JAVA\javafx\javafx-sdk-15.0.1\lib as an example is the PATH where the files I unzipped.

  1. Once you done, and everything will be fine.
