Set up the Environment of Shell Programming
2021-10-20

The first task given in first class.

Preparation

Install an SSH Client: http://mirror.linuxtrainingacademy.com/git/windows/.

(select “Use Git and optional Unix tools from the Windows Command Prompt”)

Install Vagrant Dependencies (no website, just search on google)

Install VirtualBox: https://www.virtualbox.org/wiki/Downloads.

Install Vagrant: http://mirror.linuxtrainingacademy.com/vagrant/.

(accept the agreement)

Open the Command Window

Add a Box to Vagrant: vagrant box add jasonc/centos7

Create a Working Folder: mkdir shellclass

Then get in: cd shellclass

do same thing to folder “testbox01”

Create First Vagrant Project: vagrant init jasonc/centos7

Create First Virtual Machine: vagrant up

if any change in Vagrantfile : vagrant reload

Confirm it Running: vagrant status

Connect to the Virtual Machine: vagrant ssh

then exit

then vagrant halt to stop the Virtual Machine
