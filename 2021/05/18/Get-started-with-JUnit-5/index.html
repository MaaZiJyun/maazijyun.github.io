Introduction

Junit is an open source unit test framework with annotations, which is developed by Kent Beck and Erich Gamma.

Major IDEs such as Eclipse, Netbeans, IntelliJIDEA and Blue provide JUnit integration where you can both write and run the unit tests from those IDE.
