Group information

Project title: Color it

Group name: WeDesign

Group member:

  • Aous
  • Ma
  • Wang

Project/Research information

Introduction

Designer struggling to find a color that fits their needs.so designers can select their own appropriate color groups or individual colors for use in the designing their software, or generate random colors automatically.

Objective

To help designers find the color that is inside their mind. Facilitate designers to design a more beautiful UI interface.

Background

For the designers who are not familiar with the colours code, it is hard to design and develop the website or application interface without a sample of colour code. Many people who do not know much about design and have not studied systematically find it difficult to use the right color combinations to make good-looking UI designs.

In order to help designers and developers to find the colours they want, this website will provide both code and the sample of the colours.

Methods

The website will have an option where it displays different types of colors and the user can come and choose which color fits him best.

Expected Result

The designer finds the colors for their needs

Implementation

  • Figma
  • HTML and CSS with Bootstrap framework
